# AI를 이용해 학습 계획 수립을 요청할 때  사용할 룰 
```
"학습계획을 수립해줘" 라고 하면 아래 룰을 따른다.
 1. 단계별로 나누어 학습계획을 수립
     ( 내용이 많은 경우 파일을 나눠서 생성하여 내용이 누락되거나 축소되는 일이 없더록 한다. )
 2. 단계별로 디렉토리를 구분하고 각 디렉토리별로 아래 조건을 만족시킨다.
       - 내용은 "계획 및 목표" , "학습자료" , "연습문제" , "연습문제 해답" 으로 구성하고 각각 디렉토리로 구분
       - 계획 및 목표 : md 파일로 생성하고 세부적으로 작성
       - 학습자료 : 계획 및 목표를 실행하기 위한 실직적인 학습 자료를 생성하되 단계별로 다음 단계로 넘어갈 수 있는  능력일 키울 수 있도록 세부적으로 작성
       - 연습문제 : 학습자료를 통해 학습을 진행한 이후 본인의 학습 정도를 파악할 수 있도록 연습문제를  제공
       - 연습문제 해답 : 연습문제에 대한 모범 단안을 제시  
3. 전체 학습과정을 정리한 md파일을 최상단에 생성하여 학습을 시작하기 전에 전체 를 알수 있도록 하고 선행지식일 필요한 경우 어떤 부분인지 와 학습에 필요한 준비(필요)사항도 포함한다.
4. 파일의 내용이 많을 경우 생략하지 말고 누락되는 정보 없이 그대로 파일을 나눠서 생성한다.
```
